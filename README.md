# retail-sales-profitability-analysis-with-sql-database-design

### Project Overview
Minimart is a growing retail company offering a wide range of products, including food, beverages, vegetables, and fruits across multiple branches in Indonesia. However, only a portion of branches currently have recorded sales data.

This project focuses on designing a relational database and analyzing transactional data to uncover business insights, improve operational efficiency, and support decision-making.

### Key Focus Areas
- Database design and data modeling (ERD)
- Sales performance analysis across branches
- Profit and transaction trend analysis (monthly & quarterly)
- Product performance and profitability
- Employee contribution analysis

### Tools
MySQL, DBeaver

### Output
- Structured relational database (8 tables)
- SQL-based analysis answering key business questions
- Actionable insights for sales, profit, and operational improvement
